# widget_path_mappings

Storefront widget — URL path patterns → widget config (which proactive prompt, what greeting).

**Primary key:** `id`

## Columns

| Column | Type | Nullable | Notes |
|---|---|---|---|
| `id` | `uuid` | — | PK · default: `gen_random_uuid()` |
| `workspace_id` | `uuid` | — | → [[workspaces]].id |
| `path` | `text` | — |  |
| `match_type` | `text` | — | default: `'prefix'` |
| `category` | `text` | — |  |
| `created_at` | `timestamptz` | — | default: `now()` |

## Foreign keys

**Out (this → others):**

- `workspace_id` → [[workspaces]].`id`

**In (others → this):**

_None._

## Common queries

### List rows for a workspace
```ts
const { data } = await admin.from("widget_path_mappings")
  .select("id, created_at")
  .eq("workspace_id", workspaceId)
  .order("created_at", { ascending: false }).limit(50);
```

### Count since a given time
```ts
const { count } = await admin.from("widget_path_mappings")
  .select("id", { count: "exact", head: true })
  .gte("created_at", since);
```
