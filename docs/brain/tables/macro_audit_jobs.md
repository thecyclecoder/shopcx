# macro_audit_jobs

Jobs that re-audit macro acceptance rates and flag low-performers for review.

**Primary key:** `id`

## Columns

| Column | Type | Nullable | Notes |
|---|---|---|---|
| `id` | `uuid` | — | PK · default: `gen_random_uuid()` |
| `workspace_id` | `uuid` | — | → [[workspaces]].id |
| `product_intelligence_id` | `uuid` | — | → [[product_intelligence]].id |
| `status` | `text` | — | default: `'pending'` |
| `total` | `int4` | — | default: `0` |
| `completed` | `int4` | — | default: `0` |
| `results` | `jsonb` | — | default: `'[]'` |

| `updated_at` | `timestamptz` | — | default: `now()` |

## Common queries

### List rows for a workspace
```ts
const { data } = await admin.from("macro_audit_jobs")
  .select("id, status, created_at, updated_at")
  .eq("workspace_id", workspaceId)
  .order("created_at", { ascending: false }).limit(50);
```

### Bucket by status (probe actual values first)
```ts
const { data } = await admin.from("macro_audit_jobs")
  .select("status").limit(2000);
const counts = new Map();
for (const r of data || []) counts.set(r.status, (counts.get(r.status) || 0) + 1);
```

### Count since a given time
```ts
const { count } = await admin.from("macro_audit_jobs")
  .select("id", { count: "exact", head: true })
  .gte("created_at", since);
```
