# Lifecycle — Ad Creative Agent (Dahlia)

End-to-end trace of how a fresh Meta static ad goes from raw product intelligence to a ready-to-test creative in [[../functions/growth|Bianca]]'s bin, with **no human gate** — because every claim is verifiable by construction and the render is vision-QA'd. Dahlia is a worker under Max, peer to Bianca ([[../functions/growth]]).

## The chain

1. **Cadence** — [[../inngest/ad-creative-cadence]] `adCreativeCadenceCron` (`0 11 * * *`) fans out per workspace; `dispatchAdCreativeCadence` measures each intelligence-backed product's bin depth ([[../libraries/ready-to-test]] `listReadyToTest`) and enqueues an [[../tables/agent_jobs]] `kind='ad-creative'` job (with the deficit) for any product below `DEFAULT_BIN_FLOOR` (4). Idempotent per UTC day.

2. **Claim + dispatch** — [[../libraries/builder-worker]] fills the `ad-creative` lane (`MAX_AD_CREATIVE`, deterministic Node) and `runAdCreativeJob` calls [[../libraries/creative-agent]] `runAdCreativeLoop`.

3. **Intelligence** — [[../libraries/product-intelligence]] `getProductIntelligence` fans out every `product_*` table for the product (benefits, ingredient research + citations, ad angles, review clusters, featured/transformation reviews, media, variants, store proof points, the computed offer).

4. **Angle** — [[../libraries/creative-brief]] `selectAngles` scores candidates on **two axes**: `acquisitionPower` (cold-scroll stopping power — a real transformation, a skeptic objection, curiosity) vs `retentionTruth` (energy-no-crash, taste). The commodity "no-crash" angle — the #1 review cluster but a converts-nobody cold hook — is **demoted** to supporting body copy. The top acquisition angle leads.

5. **Brief** — `buildCreativeBrief` assembles the hook + lead proof (a real review or ingredient citation) + a **single-reviewer transformation** (quote + name + photo, all the same person) + supporting retention truths + proof stack + the **offer as an allowed price treatment** (strikethrough→discount or per-serving-vs-latte — never bare MSRP) + image refs.

6. **Generate — 3-placement pack ([[../libraries/creative-pack]] Phase 2)** — [[../libraries/creative-generate]] `generateCreative` renders THREE placement-sized statics from the same brief concept: feed 4:5 (canonical), stories/reels 9:16, right-column 1:1. Same psychology, only aspect ratio varies. Meanwhile, [[../libraries/creative-pack]] `buildMetaCopyPack` produces four headline variations + four primary-text variations (one LF8 driver, varied hooks — Meta rotates them). A before/after MAY be an AI-generated **photorealistic full-body** transformation (CEO grey-area call, 2026-07-10) — but the quote+name are a real review and nothing captions the image as authentic.

7. **QC gate** — [[../libraries/creative-qa]] runs the vision pass on the canonical feed_4x5: headline exact, all text legible, no bare price, no fabricated authenticity caption, transformation photorealistic (not cartoon). Two paths keyed on the `DAHLIA_QC_MODE` env kill-switch: **`box` (default)** = `qaCreativeViaBoxSession` — a top-level `claude -p` on Max via the [[../libraries/creative-qc|creative-qc]] skill (dahlia-creative-qc-via-box-session Phases 1–3; the tmp jpeg lives at `/tmp/creative-qc-<uuid>.jpg` and is deleted after), so the lane never needs `ANTHROPIC_API_KEY`. Phase 3 / Fix 1 wraps the prompt via [[../libraries/creative-qc-sandbox]] `buildQcPrompt` (injection-safe prompt layer) and gates tool access via [[../libraries/ad-creative-qc-permission-gate]] (PreToolUse hook). **`direct`** = `qaCreative`, a direct Opus vision API call unchanged — the one-flag revert if the box path misbehaves (any other value degrades to `box`). Both paths **fail closed**: any session error, cap, timeout, undecodable image, or unparseable/non-JSON verdict yields `pass:false` so nothing unchecked reaches the bin. On `pass:false`, regenerate (up to `MAX_QA_ATTEMPTS`). The canonical passes; siblings inherit the same status.

8. **Bin insertion ([[../libraries/creative-pack]] Phase 2–3)** — `insertReadyCreative`: [[../tables/product_ad_angles]] (stamped with the 4×4 copy pack via `metadata.copy_pack`) → [[../tables/ad_campaigns]] `status='ready'` → THREE static [[../tables/ad_videos]] siblings (feed 4:5 canonical + stories/reels 9:16 + right-column 1:1), linked via `format_variant_of_id`. All three renders uploaded to the private `ad-tool` bucket ([[../libraries/ad-storage]]). Single `landing_url` = the **battle-tested Shopify PDP** `{shopify_primary_domain}/products/{handle}` (policy CEO 2026-07-10; storefront / advertorial-variant landers are a later winner-only experiment). **Completeness gate (Phase 3):** [[../libraries/creative-pack]] `isCreativePackComplete` re-checks at publish time (when Bianca calls [[../libraries/media-buyer-agent]] `publishToMeta`) — only packs with all 3 placements + ≥4 headlines + ≥4 primary texts are allowed to Meta. **LF8 gate satisfied by construction (2026-07-14):** the caption (headline + primaryText) is guaranteed to carry a [[../libraries/lf8|Life-Force-8]] term whenever the brief carries an LF8-adjacent supporting benefit, so Bianca's test loop never surfaces zero-LF8 copy to the ads-supervisor (the [[../libraries/ads-supervisor]] `live_ad_lf8_thin` gate gates on the same [[../libraries/lf8]] keywords).

9. **Handoff** — `listReadyToTest` now surfaces the campaign to Bianca's [[../libraries/media-buyer-agent]] test loop.

## North star
Dahlia optimizes a bounded proxy — *bin depth × brief quality* — and feeds Bianca; **Max owns the objective** (does the creative actually win in-market) and holds her leash. She never launches an ad herself; she stocks the bin, Bianca tests, the media-buyer grader scores realized ROAS. She is audited by the [[../libraries/ads-supervisor]] standing every-3h pass ([[../inngest/ads-supervisor-cadence]]), which checks for drift (bin depth below floor, zero seeded angles) and authors fix-specs. See [[../operational-rules.md]] § North star.
